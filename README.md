# 视频编码检测工具

一个纯前端的视频格式分析工具，无需后端服务，直接在浏览器中运行。

## 在线使用

🔗 https://miny716.github.io/analysisVideoFormat/

🔗 https://6a1e8c867dec61007bf28c13--leafy-mooncake-7fbfda.netlify.app

## 功能

- 支持输入视频 URL 进行在线分析（自动使用 HTTP Range 按需读取，无需下载完整文件）
- 支持上传本地视频文件进行分析（文件不会上传服务器，完全本地处理）
- 分析结果展示以下信息：

| 字段         | 说明                                   |
| ------------ | -------------------------------------- |
| 来源         | URL 地址或本地文件名                   |
| 容器格式     | 如 MPEG-4、Matroska 等                 |
| 文件大小     | 自动换算 B / KB / MB / GB              |
| 总时长       | HH:MM:SS 格式                          |
| 视频编码     | 如 AVC（H.264）、HEVC（H.265）等       |
| 编码 Profile | 如 High、Main、Baseline                |
| 编码 Level   | 如 4.1、5.1（需 < 5 才兼容微信等平台） |
| 分辨率       | 宽 × 高（像素）                        |
| 帧率         | fps                                    |
| 视频码率     | kbps                                   |
| 色彩空间     | 如 YUV                                 |
| 位深         | 如 8 bit、10 bit                       |
| 音频轨道数   | 包含音频编码格式                       |

- 支持展开查看完整原始 JSON 数据

## 使用方式

### 方式一：输入 URL

1. 在输入框中粘贴视频 URL
2. 点击「分析 URL 视频」
3. 等待分析完成，查看结果

> 注意：URL 对应的服务器需要支持 CORS 跨域及 HTTP Range 请求

### 方式二：上传本地文件

1. 点击「上传本地文件」Tab
2. 选择本地视频文件（支持所有常见视频格式）
3. 点击「分析本地视频」
4. 等待分析完成，查看结果

## 技术实现

- 纯原生 HTML + CSS + JavaScript，无任何框架依赖
- 视频解析基于 [mediainfo.js](https://github.com/buzz/mediainfo.js)（WebAssembly 版本的 MediaInfo）
- URL 模式通过 HTTP Range 请求按需读取文件块，mediainfo 解析完格式信息后自动停止，避免完整下载大文件
- 本地文件模式使用 File Stream API 流式读取，实时展示读取进度

## 视频资源安全性说明

### 本地文件

- **文件不会离开设备**：本地视频文件通过浏览器 File API 读取，全程在内存中处理，不会上传至任何服务器
- **无网络传输**：分析过程仅调用 WebAssembly 模块在本地计算，可在完全离线环境下使用
- **不留痕迹**：页面关闭后所有数据自动释放，不写入 localStorage 或 IndexedDB

### URL 视频

- **仅读取必要数据**：通过 HTTP Range 请求按需分块读取，通常只需读取文件头部（几百 KB）即可完成格式解析，不会下载完整视频内容
- **请求由浏览器发出**：URL 请求直接由用户浏览器发起，工具本身不经手、不转发、不存储任何视频数据
- **仅支持公开可访问的 URL**：工具无法绕过目标服务器的鉴权机制（如 Token、Cookie、IP 白名单），私有视频链接需确保当前网络环境有访问权限
- **受浏览器同源策略限制**：目标服务器需返回正确的 CORS 响应头（`Access-Control-Allow-Origin`），否则浏览器会拦截请求；**建议不要将含敏感信息的私有视频 URL 输入到不受信任的第三方部署版本中**

### 第三方依赖

| 依赖         | 来源                  | 用途                        |
| ------------ | --------------------- | --------------------------- |
| mediainfo.js | esm.sh + jsdelivr CDN | 视频格式解析（WebAssembly） |

> 如需在完全离线或内网环境使用，可将 `mediainfo.js` 和对应的 `.wasm` 文件下载到本地，修改 `index.html` 中的 CDN 地址为本地路径。
